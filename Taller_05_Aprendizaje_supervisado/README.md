# Taller 05 - Aprendizaje Supervisado

Este folder contiene los notebooks del taller de aprendizaje supervisado, enfocados en dos casos aplicados:

- Clasificación: modelo para predecir una variable categórica mediante regresión logística.
- Regresión: modelo para estimar un precio continuo a partir de características de autos usados.

## Contenido

- Clasificación - Caso aplicado Logística.ipynb
- Clasificacion_resuelto.ipynb
- Regresión - Caso aplicado Avaluo de autos.ipynb
- Regresion_resuelto.ipynb
- Dataset/ con los archivos necesarios para ejecutar los ejercicios

## Requisitos

Para ejecutar los notebooks se recomienda tener instalado:

- Python 3.9 o superior
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- jupyter

## Instrucciones

1. Abrir el notebook correspondiente en Jupyter o VS Code.
2. Ejecutar las celdas en orden.
3. Verificar que la carpeta Dataset esté disponible en la misma ruta esperada por los notebooks.
4. Si se desea, comparar los resultados entre la versión base y la versión resuelta.

## Objetivo del taller

Practicar los conceptos básicos de aprendizaje supervisado mediante dos problemas reales:

- clasificación binaria
- regresión para predicción numérica

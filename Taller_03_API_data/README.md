## Descripción de la actividad
El objetivo de este taller es integrar la consulta de datos desde APIs usando Python con la construcción de reportes en Power BI.

### Parte 1: Python
- Se realizó al menos una consulta a una API usando `requests`.
- Se convirtió la respuesta JSON a un `DataFrame`.
- Se limpiaron y transformaron los datos.
- Se generó un dataset final listo para análisis.
- Se exportaron los resultados a archivos `.csv`.

### Parte 2: Power BI
- Se cargaron los CSV en Power BI.
- Se construyó un dashboard con visualizaciones y segmentadores.
- El reporte final se guardó en `output/SPORT_RETAIL_LAM.pbix`.
